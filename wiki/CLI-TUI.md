# TUI 界面

`ppsctl watch` 启动终端交互界面(Terminal UI),基于 Bubble Tea 框架构建。

## 界面布局

```
┌─────────────────────────────────────────────────────┐
│  ┌──────────────────┐  ┌──────────────────────────┐ │
│  │    运行列表        │  │     运行详情 / 日志       │ │
│  │                   │  │                          │ │
│  │  #123 deploy ✓    │  │  流水线: deploy.yaml     │ │
│  │  #122 nightly ✗   │  │  状态: ✅ 已完成         │ │
│  │  #121 deploy ◌    │  │  耗时: 12.3s             │ │
│  │                   │  │  ── 任务详情 ──          │ │
│  │                   │  │  pull-images  ✅         │ │
│  │                   │  │  migrate       ✅         │ │
│  │                   │  │  restart       ✅         │ │
│  │                   │  │  ── 实时日志 ──          │ │
│  │                   │  │  [12:00:01] Pulling...   │ │
│  │                   │  │  [12:00:03] Done         │ │
│  └──────────────────┘  └──────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

## 操作按键

| 按键 | 功能 |
|:--|:--|
| `↑` / `↓` | 在运行列表中导航 |
| `Enter` | 查看选中运行的详情 |
| `q` / `Ctrl+C` | 退出 TUI |
| `r` | 刷新运行列表 |
| `c` | 取消选中运行 |
| `Tab` | 焦点切换(列表/详情) |
| `F1` | 帮助信息 |

## 颜色标识

- 🟢 **绿色** — 成功完成
- 🔴 **红色** — 失败
- 🟡 **黄色** — 运行中
- ⚪ **灰色** — 未开始/已取消

## 功能特性

1. **实时刷新** — 自动轮询后端,更新运行状态
2. **状态高亮** — 不同状态使用不同颜色和图标
3. **日志查看** — 选中运行后右侧显示实时日志
4. **任务详情** — 展示 DAG 中每个任务的执行状态
5. **取消操作** — 无需退出 TUI 即可取消运行
